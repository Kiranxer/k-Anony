# Telegram Anonymous Chat Bot

Anonymous 1-to-1 chat bot for Telegram with:

- Random stranger matching
- Gender setting (`/gender girl|boy|other`)
- Interests (`/interests roblox, anime, gaming`)
- Premium feature: **match only with girls** for **14 hours** for **300 Telegram Stars**

> ⚠️ All gender info is self-declared by users. Do not promise 100% accuracy.

## Requirements

- Node.js 18+
- A Telegram bot token from [@BotFather](https://t.me/BotFather)
- Bot payments enabled with **Telegram Stars** for your bot
